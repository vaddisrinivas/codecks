# Codecks Scope Review Plan

Goal: review the current app, git/GitHub state, local untracked scope, and recommend valuable new features plus improvements.

## Phases

- [complete] Establish repo/GitHub state and planning context.
- [complete] Inspect committed app architecture, docs, workflows, and recent history.
- [complete] Inspect untracked local v2 scope separately from committed code.
- [complete] Synthesize feature opportunities, improvement backlog, and risks.

## Acceptance Checks

- Current PR/issue state checked with `gh`.
- Current branch, remote, dirty state checked with `git`.
- Recommendations distinguish shipped/committed code from local untracked work.
- Final answer includes prioritized feature adds, improvements, and app scope.
