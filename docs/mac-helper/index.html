Codecks Mac Helper

Fallback page. The Android app should normally find your Mac and install the SSH key itself. Use this page only when pairing gets stuck.

1. Enable Remote Login

On your Mac: System Settings → General → Sharing → Remote Login → On.

2. Show your Mac fingerprint

Copy this into Terminal. Compare the output with Codecks before trusting.

node -e 'const {execSync}=require("child_process"); for (const f of ["/etc/ssh/ssh_host_ed25519_key.pub","/etc/ssh/ssh_host_rsa_key.pub"]) { try { console.log(execSync(`ssh-keygen -lf ${f}`, {encoding:"utf8"}).trim()) } catch {} }'

3. Manual key install fallback

Prefer the app button: Settings → Connections → Pair Mac. If that fails, paste the public key from Codecks here to generate a Terminal command.
