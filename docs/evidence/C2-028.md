- `android/transport/ssh/build.gradle.kts`
  - Adds isolated `:transport:ssh` module.
  - Keeps SSH dependency out of domain, data, feature, and design modules.
- `android/transport/ssh/src/main/kotlin/io/codecks/transport/ssh/SshjCommandTransport.kt`
  - Connects with pinned host fingerprint.
  - Authenticates with private key material.
  - Executes one command with timeout and bounded output capture.
  - Maps auth, host-key, timeout, and connection failures into typed safe failures.
- `android/transport/ssh/src/main/kotlin/io/codecks/transport/ssh/SshPublicKeyInstaller.kt`
  - Uses SSHJ password auth for one-time public-key installation.
  - Reuses pinned host fingerprint verification before password auth.

  - Proves output capture stores only up to the shared byte budget while draining input.

  - Proves safe authorized-keys command construction for OpenSSH public keys.
