# C2-029 Evidence

## Status

Current status: partial, Android Keystore-backed SSH key generation/provider exists and first-run setup exposes display/copy plus one-time password-assisted install. Physical leakage proof remains.

## Implemented Evidence

- `android/core/security/src/main/kotlin/io/codecks/core/security/SensitiveChars.kt`
  - Copies inbound sensitive char arrays.
  - Redacts `toString()`.
  - Clears internal char buffer on close.
- `android/core/security/src/test/kotlin/io/codecks/core/security/SensitiveCharsTest.kt`
  - Proves secrets are not printed through `toString()`.
  - Proves source arrays are copied before mutation.
  - Proves closed values cannot be read.
- `android/transport/ssh/src/main/kotlin/io/codecks/transport/ssh/SshCredentials.kt`
  - Defines `SshCredentialProvider`.
  - Supports PEM credentials for tests and `KeyPair` credentials for non-exportable Android Keystore keys.
- `android/data/targets/src/main/kotlin/io/codecks/data/targets/AndroidKeystoreSshKeyStore.kt`
  - Creates RSA signing keys in `AndroidKeyStore`.
  - Uses `PURPOSE_SIGN`/`PURPOSE_VERIFY`, SHA-256/SHA-512 digests, and RSA PKCS#1 signature padding.
  - Returns a `KeyPair` without exporting private key bytes.
- `android/data/targets/src/main/kotlin/io/codecks/data/targets/AndroidKeystoreSshCredentialProvider.kt`
  - Implements `SshCredentialProvider` using the Android Keystore.
  - Exposes public-key retrieval and rotation seams.
- `android/data/targets/src/main/kotlin/io/codecks/data/targets/OpenSshPublicKeyEncoder.kt`
  - Encodes RSA public keys as OpenSSH `ssh-rsa` public-key lines.
- `android/data/targets/src/test/kotlin/io/codecks/data/targets/OpenSshPublicKeyEncoderTest.kt`
  - Proves generated RSA public keys encode to an OpenSSH-compatible wire payload.
  - Proves public-key comments cannot inject new lines.
- `android/app/src/main/java/io/codecks/app/data/CodecksDataContainer.kt`
  - Wires the first Finder proof through a credential-provider seam.
  - Uses `AndroidKeystoreSshCredentialProvider` for SSH action execution.
- `android/transport/ssh/src/main/kotlin/io/codecks/transport/ssh/SshPublicKeyInstaller.kt`
  - Installs the phone public key with one-time SSH password auth.
  - Appends only the public key to `~/.ssh/authorized_keys`.
  - Returns typed install results without exposing the password.
- `android/transport/ssh/src/test/kotlin/io/codecks/transport/ssh/SshjPublicKeyInstallerTest.kt`
  - Proves install command quotes the OpenSSH public key.
  - Rejects non-OpenSSH key payloads before command construction.
- `android/feature/connection/src/main/java/io/codecks/feature/connection/ConnectionSetupScreen.kt`
  - Shows generated phone public key.
  - Supports copy/manual-installed fallback.
  - Captures a password for one install attempt and tells the user it is not saved.
- `android/app/src/main/java/io/codecks/app/ui/CodecksApp.kt`
  - Generates the phone public key through `AndroidKeystoreSshCredentialProvider`.
  - Copies only the public key to the Android clipboard.
  - Wraps install password in `SensitiveChars` and clears it after the suspend call scope.
- Verification run:
  - `ANDROID_HOME=$ANDROID_HOME ./gradlew assembleLocalDebug assembleIncubatorDebug :domain:targets:test :domain:decks:test :runtime:actions:test :data:decks:testDebugUnitTest :data:receipts:testDebugUnitTest :data:targets:testDebugUnitTest :core:security:test :transport:ssh:test checkReleaseBoundary`
  - `./tools/check-local-release-boundary.sh`
